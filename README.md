# Hospital Management System

A console-based Hospital Management System developed in C++ using Object-Oriented Programming (OOP) concepts and File Handling.

## Features

* Add new patients
* View patient records
* Delete patient records
* Add doctors
* View doctor records
* Delete doctor records
* Book appointments
* View appointment details
* Validate patient and doctor IDs
* Persistent data storage using text files

## Technologies Used

* C++
* Object-Oriented Programming
* File Handling
* Classes and Objects
* Conditional Statements
* Loops

## Project Structure

* patients.txt – Stores patient records
* doctors.txt – Stores doctor records
* appointments.txt – Stores appointment details

## How to Run

1. Clone the repository
2. Compile the code

g++ main.cpp -o hospital

3. Run the executable

./hospital
